## 📝 <span id="description">Description</span>

### Aperçu du projet :

Showcaser est une plateforme moderne de création et de personnalisation de templates. Elle permet aux utilisateurs de créer, modifier et partager des modèles personnalisés pour divers usages, le tout avec une interface intuitive et élégante.

### Objectifs et portée :

Le projet vise à simplifier le processus de création de templates en offrant une solution tout-en-un qui combine :

- Une interface de création visuelle intuitive
- Des options de personnalisation avancées
- Un système de prévisualisation en temps réel
- Une gestion des templates avec possibilité de partage

### Caractéristiques principales :

- Éditeur visuel moderne
- Prévisualisation en direct
- Gestion des templates
- Interface responsive
- Intégration avec Stripe pour les fonctionnalités premium

## 💻 <span id="utilisation">Utilisation</span>

1. Clonez le repository
2. Installez les dépendances :

```bash
npm install
# ou
yarn install
# ou
pnpm install
```

3. Lancez le serveur de développement :

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
```

4. Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur
